# Inzich

A Python terminal app that fetches financial data for any publicly traded company and exports a professional Excel workbook with an AI-generated research summary.

![Inzich Demo](docs/demo.png)

---

## What It Does

Enter any company name — Inzich resolves it to a confirmed ticker, fetches up to 4 years (or 4 quarters) of financial data from Yahoo Finance, and exports two files to your `exports/` folder:

- **`{TICKER}_{DATE}.xlsx`** — a 6-sheet Excel workbook covering company overview, historical prices, income statement, balance sheet, cash flow statement, and financial ratios
- **`{TICKER}_{DATE}_ai_summary.txt`** — an AI-generated equity research summary with cited sources, powered by OpenAI

Both files open automatically after export.

---

## How It Works

```
Launch app
  ↓
Enter a company name (free text — no ticker needed)
  ↓
OpenAI validates → "Did you mean Apple Inc. (AAPL)?"
  ↓
Select period type: Annual or Quarterly
  ↓
Select number of periods: 1–4
  ↓
App fetches all data from Yahoo Finance
  ↓
Excel workbook + AI summary .txt exported and auto-opened
  ↓
"Analyze another company?" → loop or exit
```

Company name resolution handles misspellings, gibberish, multiple companies entered at once, and companies that are not publicly traded — with clear error messages for each case.

## Installation

### Step 1 — Clone the repository

```bash
git clone https://github.com/knbeltz/inzich.git
cd inzich
```

### Step 2 — Create a virtual environment

**Windows:**
```
python -m venv .venv
.venv\Scripts\activate
```

**Mac:**
```
python3 -m venv .venv
source .venv/bin/activate
```

You'll know it's active when your terminal prompt shows `(.venv)`.

### Step 3 — Install dependencies

**Windows:**
```
pip install -r requirements.txt
```

**Mac:**
```
pip3 install -r requirements.txt
```

### Step 4 — Set up your environment variables

Copy the example file and fill in your values:

**Windows:**
```
copy .env.example .env
```

**Mac:**
```
cp .env.example .env
```

Then open `.env` and add your OpenAI API key:

```
OPENAI_API_KEY=your_key_here
DB_PATH=./data/cache.db
EXPORT_DIR=./exports
```

You can get an OpenAI API key at [platform.openai.com](https://platform.openai.com).

### Step 5 — Run the app

**Windows:**
```
python main.py
```

**Mac:**
```
python3 main.py
```

## What Gets Exported

Each run produces two files in the `exports/` folder:

### Excel Workbook (`{TICKER}_{DATE}.xlsx`)

| Sheet | Contents |
|---|---|
| Company Overview | Ticker, sector, market cap, price, employees, and more |
| Historical Prices | Daily OHLCV data for the selected period |
| Income Statement | Revenue, gross profit, EBITDA, net income, EPS |
| Balance Sheet | Cash, assets, liabilities, shareholders equity |
| Cash Flow Statement | Operating, investing, financing cash flows and free cash flow |
| Financial Ratios | P/E, ROE, ROA, margins, debt/equity, and more |

### AI Summary (`{TICKER}_{DATE}_ai_summary.txt`)

An equity research summary generated by OpenAI using live web search. Includes cited sources.
