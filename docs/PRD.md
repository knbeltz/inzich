# Inzich V1 — Product Requirements Document

## Overview

Inzich is a Python terminal application that allows users to generate a professional Excel financial analysis workbook for any publicly traded company. The user enters a company name or ticker, the app fetches data from Yahoo Finance, and exports a structured multi-sheet Excel file to disk.

**V1 is a single workflow:** one company, one report, one export.

---

## Target User

Finance students, analysts, and hobbyists who want clean, structured financial data for a company without manually downloading from multiple sources.

---

## V1 User Flow

1. User launches the app (`python main.py`)
2. Splash screen appears
3. Main menu shows: "Company Information" or "Quit"
4. User selects "Company Information"
5. User types a company name or ticker (e.g., "Apple" or "AAPL")
6. App searches Yahoo Finance and returns matching results
7. User confirms the correct company
8. User selects which sheets to include:
   - Everything (all sheets), OR
   - Individual selection from the 7 sheet categories
9. App fetches data, parses it, and exports a `.xlsx` file to `exports/`
10. App confirms the file path and loops back to the main menu

---

---

## Data Source

**Primary:** Yahoo Finance via the `yfinance` Python library. No API key required.

**Optional:** OpenAI API for Sheet 7 AI summary only.

---

## Caching

SQLite database at `DB_PATH` (default: `./data/cache.db`). Per-ticker cache with 24-hour TTL. Prevents redundant Yahoo Finance calls when the same company is requested multiple times in one day.

---

## File Export

Exported files are saved to `EXPORT_DIR` (default: `./exports/`).

Filename format: `{TICKER}_{YYYY-MM-DD}.xlsx`

Example: `AAPL_2026-06-23.xlsx`

---

## Out of Scope for V1

The following are explicitly deferred to V2:

- DCF (Discounted Cash Flow) calculations
- Comparable company analysis
- FRED API integration (treasury yields, risk-free rate)
- SEC EDGAR integration (XBRL financial filings)
- WACC calculations
- Multi-company reports
- Web or GUI interface
